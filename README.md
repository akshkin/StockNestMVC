# StockNest - Inventory & Stock Management System  
A full‑stack inventory management application built with **ASP.NET Core MVC**, **Entity Framework Core**, **PostgreSQL**, and a **React frontend** with authentication, dashboards, metrics

## Features

### Authentication & Authorization
- Secure cookie‑based authentication  
- Login, logout 

### Inventory Management
- Create, update, and delete items  
- Grouping and categorization  
- Ownership tracking 
- EF Core migrations for schema evolution
- Statistics

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React, Chart.js, Redux Toolkit Query |
| Backend | ASP.NET Core MVC |
| Database | PostgreSQL (Neon) |
| ORM | Entity Framework Core |
| Auth | Cookie‑based authentication |
| Deployment | Render |
| Language | C#, TypeScript |

## 🧠 Lessons Learned

This project involved solving challenges:
- Cookie authentication in production  
- CORS and environment‑based configuration   

These experiences shaped the final architecture and improved reliability.
